1. Блок идентификатор типа файла ([Регистрация идентификаторов файла.md](./Регистрация%20идентификаторов%20файла.md)):
```C
static const TypeInfo_t FileTypeIdType =
{
	.Type = Struct, .Name = "FileTypeId", .Dims = { 0, NULL }, .Childs =
	{
		.Count = 2,
		.Item = (TypeInfo_t[])
		{
			{ Int16, "Type" },
			{ Int16, "Version" },
		}
	}
};

typedef struct
{
	uint16_t Type;
	uint16_t Version;
} FileTypeId_t;
```
